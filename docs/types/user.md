
# Информация о пользователе

```
User
{
    id:         integer, // Внутренний id пользователя в системе
    first_name: string,  // Имя пользователя.
    last_name:  string,  // Фамилия пользователя.
    email:      string,  // Адрес электонной почты.
    is_guest:   boolean, // Является ли пользователь гостем.
} 
```
